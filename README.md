
## Getting Started

# **Include the GSAP plugin script**

   This project uses the GSAP Plugin - Motion Path Plugin, make sure to include the link as follows:
   <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/MotionPathPlugin.min.js"></script>
   ```


To run this application locally, follow these steps:

1. **Install dependencies**

   ```
   npm install
   ```

2. **Start the development server**

   ```
   npm run dev
   ```



This will start the app in development mode. Open your browser and go to the URL provided in your terminal (usually `http://localhost:5173` or similar) to view the app.
